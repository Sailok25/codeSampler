-------------------
| EL PSEUDOCODIGO |
-------------------

Definición:
El pseudocódigo es una forma de escribir instrucciones de programación en un lenguaje más humano y cercano
al lenguaje natural que a un lenguaje de programación real como Java o C++. Se utiliza para planificar y 
diseñar algoritmos antes de escribir el código en un lenguaje de programación específico.


Caracteristicas:
- No sigue una sintaxis estricta como un lenguaje de programación real, lo que lo 
hace más comprensible para las personas.

- Se utiliza para representar de manera clara y lógica el flujo de un algoritmo.

- No es ejecutable directamente por una computadora, pero se puede convertir en 
código real una vez que se ha planificado adecuadamente.


Ejemplo:
Supongamos que quieres escribir un programa que sume dos números. 
Aquí tienes un ejemplo simple de cómo se vería en pseudocódigo:
-------------------------------------------
Inicio
  // Pedir al usuario dos números
  Escribir "Ingresa el primer número: "
  Leer numero1
  Escribir "Ingresa el segundo número: "
  Leer numero2

  // Calcular la suma de los números
  suma = numero1 + numero2

  // Mostrar el resultado
  Escribir "La suma es: ", suma
Fin
---------------------------------------------
